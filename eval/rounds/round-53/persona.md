# Round 53 Persona

**Role:** 대학교 입학처 데이터 분석팀장 (University Admissions Data Analytics Team Lead)
**Institution:** 연세대학교 (Yonsei University)

**Background:** 15년간 대학 행정업무 경험, 최근 3년간 데이터 분석 업무 담당. Excel과 SPSS는 잘 다루지만 디자인 툴은 서툼. 학내 정치적 맥락을 잘 이해하고 있어서 어떤 메시지가 의사결정자들에게 어필할지 알고 있음.
