🔀 API Gateway

Single entry point. Handles authentication, rate limiting (1000 req/min), request routing, and response aggregation. Deployed as Kong or custom Express middleware.

⚡ Event-Driven

Services communicate asynchronously via RabbitMQ. Order events trigger notifications, inventory updates, and analytics — fully decoupled.

💾 Polyglot Persistence

PostgreSQL for relational data (ACID), Redis for caching & sessions (sub-ms reads), MongoDB for flexible document storage and audit logs.

🚀 Deployment

Each service is containerized (Docker), orchestrated via Kubernetes, with Helm charts for config. CI/CD via GitHub Actions. Blue-green deployments.
