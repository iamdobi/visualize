API Gateway

Kong / Express gateway handles rate limiting, JWT validation, request routing, and load balancing. Single entry point for all clients with built-in circuit breaking.

⚡

Event-Driven

Services communicate asynchronously via RabbitMQ. Order creation publishes events consumed by Notification, Inventory, and Analytics — fully decoupled and resilient.

🗄️

Polyglot Persistence

Each service owns its data store. PostgreSQL for relational data, MongoDB for flexible catalogs, Redis for caching and sessions — optimized per use case.

🚀

Deployment

Containerized with Docker, orchestrated on Kubernetes. CI/CD via GitHub Actions with blue-green deployments. Observability through Prometheus, Grafana, and Jaeger tracing.
