Architecture Notes

🔄 Event-Driven

Services communicate asynchronously via RabbitMQ message queues, ensuring loose coupling and resilience.

📈 Auto-Scaling

Each microservice scales independently based on load. Kubernetes manages container orchestration.

🛡️ Security Layers

JWT authentication at the gateway, encrypted data at rest, and mTLS between services.

⚡ Caching Strategy

Redis handles session data and hot product queries. Elasticsearch powers full-text search with sub-50ms latency.
