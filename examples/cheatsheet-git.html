Repository
Setup
  • git init Initialize new repo
  • git clone <url> Clone remote repo
  • git config user.name "N" Set name
  • git config user.email "E" Set email
  • git config --global -l List config
Basic Workflow
  • git status Show working tree status
  • git add . Stage all changes
  • git add <file> Stage specific file
  • git commit -m "msg" Commit staged changes
  • git log --oneline Compact commit history
  • git diff Show unstaged changes
  • git diff --staged Show staged changes
Branching
  • git branch List branches
  • git branch <name> Create branch
  • git checkout <name> Switch branch
  • git switch -c <name> Create & switch
  • git branch -d <name> Delete branch
  • git branch -a List all (incl. remote)
Merging
  • git merge <branch> Merge into current
  • git merge --no-ff <b> Merge with commit
  • git merge --abort Abort merge
  • git mergetool Open merge tool
  • git log --merge Show conflicting commits
Remote
  • git remote -v List remotes
  • git remote add origin <u> Add remote
  • git push -u origin main Push & set upstream
  • git pull Fetch & merge
  • git fetch Download without merge
  • git push --force-with-lease Safe force push
Undoing Changes
  • git restore <file> Discard changes
  • git restore --staged <f> Unstage file
  • git reset HEAD~1 Undo last commit (keep)
  • git reset --hard HEAD~1 Undo & discard
  • git revert <hash> Reverse a commit
  • git clean -fd Remove untracked files
Stashing
  • git stash Save dirty working dir
  • git stash pop Apply & remove stash
  • git stash list List all stashes
  • git stash apply Apply without removing
  • git stash drop Delete latest stash
  • git stash -m "msg" Stash with message
Advanced
  • git rebase main Rebase onto main
  • git rebase -i HEAD~3 Interactive rebase (last 3)
  • git cherry-pick <hash> Apply specific commit
  • git bisect start Binary search for bugs
  • git reflog History of HEAD changes
  • git tag v1.0 Create lightweight tag
  • git tag -a v1.0 -m "msg" Annotated tag
  • git blame <file> Show who changed what
  • git log --graph --all Visual branch graph
  • git worktree add ../fix b Multiple working trees
